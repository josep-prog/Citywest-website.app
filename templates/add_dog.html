Note: when you use this page to add the dog, some attributes are required:

  • User ID: it corresponds to the user email address.
  • Name: name of the dog.
  • Alteration Status: whether the dog is spayed/neutered when it is surrendered.
  • Age: the age of the dog when it is surrendered.
  • Description: this include a description of a dog, any health concerns or other details.
  • Alteration Status: whether the dog is spayed/neutered when it is surrendered.
  • Surrender Date: the date when this dog is surrendered to the shelter.
  • Surrender Reason: the reason is required.
  • Surrender By Animal Control: a flag variable whether this dog is surrendered by animal control department.
